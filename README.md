# Description

# Requirements

- Installed NodeJS (tested with NodeJS v14+)
- Installed Yarn

- Installed node modules:

```
yarn install

```

# Quick Start tests

```
yarn start-sandbox

```

```
yarn migratefa2

```
This command will deploy a fa2 token contract. It may not start the first time. Wait 5-30 seconds or spam command


```
yarn test

```

```
yarn stop-sandbox

```

# Compile contract

```
yarn compile

```

# Deploy contract

```
yarn migrate

```
